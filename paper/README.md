# Paper Build

This folder contains the LaTeX source for the dissertation paper.

## Structure
- `paper/tex/main.tex` – main entry point
- `paper/tex/sections/` – section files
- `paper/tex/tables/` – LaTeX tables
- `paper/tex/figures/` – figure assets extracted from the original PDF
- `paper/tex/refs.bib` – bibliography
- `paper/tex/macros.tex` – custom macros

## Build
From repo root:
```bash
./paper/build.sh
```

Output PDF:
- `paper/final/final.pdf`
